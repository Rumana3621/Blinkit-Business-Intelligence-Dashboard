# 🛒 Blinkit Business Intelligence Dashboard (Power BI)

## 📌 Project Overview

This project is an interactive Business Intelligence dashboard developed in **Power BI** to analyze Blinkit's sales performance, customer behavior, product categories, brands, and delivery status. The dashboard provides valuable business insights through dynamic visualizations and interactive filters.

---

## 🎯 Objectives

- Analyze total sales and order performance
- Track monthly sales trends
- Identify top-selling product categories and brands
- Monitor delivery status performance
- Provide interactive filtering for business analysis

---

## 🛠️ Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Microsoft Excel
- Git & GitHub

---

## 📊 Dashboard Features

### KPI Cards
- Total Sales
- Total Orders
- Total Items Sold
- Average Order Value

### Interactive Slicers
- Category
- Customer Type
- Brand

### Visualizations
- Sales by Category
- Sales by Brand
- Top Products by Sales
- Monthly Sales Trend
- Sales by Delivery Status

---

## 💡 Business Insights

- Identified the highest revenue-generating product categories.
- Analyzed brand-wise sales performance.
- Tracked monthly sales trends.
- Evaluated delivery performance.
- Enabled interactive business analysis using slicers.
